Sorter

I used a bubble sort algorithm to sort a list of names alphabetically. I used Go as the language and GitHub Actions for the CI/CD workflow to automatically deploy new releases.

    Text Input Example:

  • Wishek
  • Davilla
  • Vanalstyne
  • Tiesiding
  • Honaunau
  • Volin
  • Caledonia
  • Monon
  • Loraine
  • Renville

    Sorted Output Example:

  • Caledonia
  • Davilla
  • Honaunau
  • Loraine
  • Monon
  • Renville
  • Tiesiding
  • Vanalstyne
  • Volin
  • Wishek
